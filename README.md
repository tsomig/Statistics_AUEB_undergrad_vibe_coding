## Για τον/την διδάσκοντα/ουσα

Στον φάκελο `instructor/`:

- **`generate_truth.py`** — παραγωγή ground-truth για όλη την τάξη
- **`auto_grader.py`** — αυτόματος βαθμολογητής

```bash
# 1. Φτιάξτε λίστα ΑΜ
cat > students.txt <<EOF
1250072
1250073
1250074
EOF

# 2. Παραγωγή σωστών απαντήσεων
python instructor/generate_truth.py --batch students.txt --outdir ./truth

# 3. Μετά την παραλαβή υποβολών στο submissions/
python instructor/auto_grader.py \
    --truth ./truth \
    --submissions ./submissions \
    --output grading.csv
```

## Τεχνικές λεπτομέρειες

- **Σπόρος**: SHA-256(AM) → ντετερμινιστικά μοναδικά δεδομένα ανά φοιτητή.
- **Ανοχές βαθμολόγησης**: 5% σχετικά για ροπές, 0.02 απολύτως για πιθανότητες, ±1 για πληθάριθμους.
- **Pass criterion**: ≥ 12 από 16 αριθμητικές απαντήσεις σωστές, ΚΑΙ υπόμνημα ≥ 250 λέξεων με ≥ 3 αναφορές σε προσωπικούς αριθμούς.
