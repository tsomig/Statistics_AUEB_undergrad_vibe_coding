## 📋 Τι κάνετε

1. **Διαβάζετε** την εκφώνηση (`assignment.pdf`).
2. **Ανοίγετε** το `student_notebook.ipynb` στο Colab.
3. Ανεβάζετε το `diafaneia_data.py` στο Colab.
4. **Πληκτρολογείτε** τον ΑΜ σας στο πρώτο κελί.
5. **Λύνετε** τα 5 ερωτήματα μέσα στο notebook (επιτρέπεται η χρήση LLM ως βοήθημα — δείτε **Παράρτημα Α'** της εκφώνησης).
6. **Γράφετε** ξεχωριστά ένα μονοσέλιδο υπόμνημα προς τον/την Υπουργό σε τελική μορφή pdf όπου στον τίτλο θα αναγράφετε τον ΑΜ σας. Π.χ. Minister_Report_00985858.
7. **Παραδίδετε** στο eClass τρία αρχεία:
   - `{ΑΜ}_submission.json` (παράγεται αυτόματα από το notebook)
   - `{ΑΜ}_memo.pdf` (το υπόμνημά σας)
   - `{ΑΜ}_notebook.ipynb` (το εκτελεσμένο notebook)

Επιτρέπεται η χρήση LLM (ChatGPT, Claude, κ.λπ.) ως βοήθημα στη συγγραφή κώδικα. **Δεν επιτρέπεται** να γράφεται από LLM το υπόμνημα.
