## 📋 Τι κάνετε

1. **Διαβάζετε** την εκφώνηση (`assignment.pdf`).
2. **Ανοίγετε** το `student_notebook.ipynb` στο Colab.
3. **Πληκτρολογείτε** τον ΑΜ σας στο πρώτο κελί.
4. **Λύνετε** τα 5 ερωτήματα μέσα στο notebook (επιτρέπεται η χρήση \en{LLM} ως βοήθημα — δείτε **Παράρτημα Α'** της εκφώνησης).
5. **Γράφετε** ξεχωριστά ένα μονοσέλιδο υπόμνημα προς τον/την Υπουργό (Word ή \en{LaTeX}).
6. **Παραδίδετε** στο \en{eClass} τρία αρχεία:
   - `{ΑΜ}_submission.json` (παράγεται αυτόματα από το notebook)
   - `{ΑΜ}_memo.pdf` (το υπόμνημά σας)
   - `{ΑΜ}_notebook.ipynb` (το εκτελεσμένο notebook)
